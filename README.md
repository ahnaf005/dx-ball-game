# DX Ball Game

A canvas-based DX Ball / Breakout game built with React and Vite.

## Live Deployment

https://simple-dx-ball-game.vercel.app/

## Features

- Paddle control with keyboard and pointer input
- Ball physics with paddle-angle reflection
- Brick collision, scoring, and restart flow
- Appwrite-backed score saving with player names
- Top 3 leaderboard

## Development

```bash
npm install
npm run dev -- --host
```
